# 🚀 SkillPilot AI

SkillPilot AI is an intelligent learning dashboard designed to help tech students learn faster, build projects, and track their progress.

It acts as a **personal AI learning copilot**, providing explanations, beginner-friendly projects, learning roadmaps, and skill tracking in one place.

---

## ✨ Features

### 💬 Ask AI
Students can ask tech questions and receive:
- Clear explanations
- Real-world examples
- Code snippets
- Interactive quizzes

### 💡 AI Project Generator
Generate beginner-friendly project ideas based on your learning track.

Tracks include:
- Software Development
- Data Analysis
- Product Design
- Cybersecurity

Each project includes:
- Description
- Difficulty level
- Estimated completion time
- Key features
- Technologies used

### 🗺️ Learning Roadmap
Generate a **4-month learning roadmap** for a specific tech career with:
- Monthly learning goals
- Required skills
- Learning resources
- Practice projects

### 📚 History
Automatically saves your generated:
- AI explanations
- Projects
- Roadmaps

So you can revisit them anytime.

### 📈 Skill Progress Tracker
Track the skills you are learning and monitor your progress visually.

---

## 🧠 Purpose

SkillPilot AI was built to solve a common problem:

> Many tech learners struggle to know **what to learn next, what projects to build, and how to track progress.**

SkillPilot AI solves this by combining:
- AI tutoring
- Project-based learning
- Career roadmaps
- Skill tracking

into a **single dashboard.**
